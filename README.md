"# Juno Knowledge Base\n\n  This repo powers Juno's conversational reasoning brain. Files here are synced hourly into JunoTalk.\n\n  ## Folder Structure\n\n  ### `osint/` — Open-source intelligence facts\n  Array of objects with:\n  ```json\n  { \"q\": \"question or topic\", \"a\": \"answer\", \"category\": \"string\", \"tags\": [\"tag\"], \"confidence\": 0.9 }\n  ```\n\n  ### `neo4j/` — Entity relationships (knowledge graph)\n  Array of objects with:\n  ```json\n  { \"subject\": \"Entity\", \"predicate\": \"relationship\", \"object\": \"description\", \"category\": \"string\", \"tags\": [\"tag\"] }\n  ```\n\n  ### `wikidata/` — Structured entity facts\n  Array of objects with:\n  ```json\n  { \"label\": \"Entity name\", \"description\": \"What it is\", \"instanceOf\": \"type\", \"category\": \"string\", \"tags\": [\"tag\"], \"claims\": { \"property\": \"value\" } }\n  ```\n\n  ## How it works\n  1. Add or edit JSON files in any folder\n  2. JunoTalk syncs every hour automatically\n  3. Juno uses the knowledge when answering relevant questions in conversation\n  "
